# qdex CLI

Terminal client for humans, bots and ops.

Planned commands:

```bash
qdex markets
qdex ticker QI-QUAI
qdex book QI-QUAI
qdex balance
qdex order buy QI-QUAI --amount 1000 --price 0.123
qdex order sell QI-QUAI --amount 500 --market --slippage-bps 50
qdex cancel --all
qdex stream fills
qdex proof trade <trade-id>
```
